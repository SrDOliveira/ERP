# NEXUM ERP - SISTEMA DE GESTAO DE LOJAS, DISTRIBUIDORAS, PEQUENAS EMPRESAS

O SOFTWARE FOI DESENVOLVIDO POR INTEIRO NO LINUX DEBIAN 13, POR ISSO, SIGA AS INSTRUÇÕES DO "COMO RODAR" PARA CRIAR A VIRTUALVENV E INSTALAR AS DEPENDÊNCIAS CONFORME O SEU S.O.


## Tecnologias
Python, Django, HTML, HTML/CSS, SQLite

## Como rodar
1. Clone
2. Criar virtualenv
3. Instalar dependências (pip install -r requirements.txt)
4. rodar `python manage.py runserver`

## Funcionalidades
- Venda:
  1. Habilite a tela PDV para ter a visão de caixa e vender os produtos conforme o QR CODE cadastrado;
  2. Emissão de cupom não fiscal;
  3. Controle de saídas e entradas;
  4. Controle de despesas;
- Cadastro de clientes;
  1. Tenha uma visão dinâmica dos clientes com maior e menor frequência de compra/atividade;

- Controle de estoque:
  1. Veja, na aba de estoque, os itens que estão prejudicando a sua empresa com maior e menor rotatividade;
  2. Veja o controle de entradas e saídas;
  3. QR CODE automático: no cadastro dos seus produtos, extraia páginas com QR CODE automático para imprimir e colar nos produtos físicos.

- Financeiro:
  1. Tenha uma visão completa com relatórios em PDFs automáticos inclusive com o seu DRE - Demonstrativo de Resultado do Exercício.

- Visão ERP:
  1. Inspirado no ERP SAP, pesquise pela barra de funções conforme instruções para ir diretamente à função desejada.
